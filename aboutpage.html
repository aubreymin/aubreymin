I use a different approach to User Research in each project or case study. I used different methods that I found effective for a case study. Every project are unique, and therefore there is no cookie cutter mold for all. Although there similarities; Including in these methods are creating a Proto Persona, collecting data through interviews and surveys. Using the data gathered, I analyze this information using an Affinity Diagram
The data I gathered from the users including their emotions to each presented task and question tells me a story. Defining our users through an Empathy Map, Journey Map, Storyboard  and User Persona - I can see the bigger picture or a User Insight. Then we composed our Problem Statement and a plan to tackle these problems.
Interviews are vital to our research. It gives us a deeper understanding of individuals of different ages and backgrounds. Depending on who is our target user is,  we selected multiple participants. We cover more diverse interviewees when our target user is in different age groups and backgrounds.
